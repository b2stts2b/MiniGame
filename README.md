# MiniGame
Applikation med minispel och highscore. I kommandotolken och grafiskt.

För att nollställa topplistan:
	- I textfiler->password.txt: ändra det som står efter 'lösenord:'.
	Använd sedan detta lösenordet för att nollställa topplistan.
Ljudinformation:
	- För att få ljud på Space Invaders måste detta spelas
	före de andra grafiska spelen. Ljud i Space Invaders finns bara 1 gång.
	- P.g.a. problem med ljuduppspelning i exe-filen byttes 
	bakgrundsmusiken i Corona från mp3 till wav. (Möjligen något sämre kvalitet)
	- Om du vill spela Corona med mp3-ljud. Ändra "wav" till "mp3" på rad 922
	i orginalfilen och kör filen från terminalen (Kräver Python).
